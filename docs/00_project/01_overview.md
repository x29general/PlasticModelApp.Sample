# サービス概要

---

## 1. はじめに

**PlasticModelApp**は、プラモデル製作を強力に支援することを目指すWebアプリケーションです。プラモデル製作に対するUXを向上させ、創造性と製作効率を高め、ユーザーがより良い作品を生み出せるようサポートします。

---

## 2. 課題と目標

### 2.1. 課題

- 塗料選びの難しさ: 多種多様な塗料が存在し、情報が分散しているため、具体的にどの商品を採用するかの判断が難しい。
- 所有塗料の管理不足: 所有している塗料の在庫や使用履歴を把握しづらく、効率的な製作が妨げられる。
- 調色の複雑さ: 調色レシピの保存や共有が難しく、再現性のある色を作るのが困難。
- 進捗管理の非効率性: プロジェクトの進捗やタスク管理が煩雑で、製作の効率が低下する。目標（展示会など）までに仕上げるのが難しい。

### 2.2. 目標

- ユーザーフレンドリーな塗料検索: 目的の塗料を迅速に見つけられる検索機能の提供。
- 効率的な塗料管理: 所有塗料の在庫管理と新しい塗料の追加を簡単に行える機能の実装。
- 調色支援機能: 調色レシピの保存、共有、シミュレーション機能の提供。
- プロジェクト管理機能: プロジェクトの進捗とタスクを効率的に管理できる機能の提供。

#### 🧩 課題

- 塗料の種類が多すぎて、どれを選べば良いか分からない。

## 4. 競合分析

### 4.1. 主な競合サービス: Hobby Japan DB

- 強み： 包括的で信頼性が高い
- 弱み： フィルタリングオプションが不十分、検索結果の並び替えが限定的、UIがやや古い

### 4.2. 他の塗料検索サービス

- 強み： 軽量でシンプルな構成
- 弱み： 新製品・廃盤情報の更新が不明瞭、持続性に懸念
