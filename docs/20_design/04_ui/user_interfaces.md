# UI

## 画面一覧

| ID | 画面名 | 概要 | 主な操作 |
| --- | --- | --- | --- |
| SC001 | トップ | アプリケーションのトップページ | 各機能へのナビゲーション |
| SC002 | 塗料検索 | 塗料を検索・閲覧する画面 | 検索、フィルタリング、塗料詳細の表示 |
| SC003 | 塗料詳細 | 選択した塗料の詳細情報を表示 | 塗料情報の閲覧、メモの追加、比較リストへの追加 |
| SC004 | 比較リスト | 選択した塗料を比較する画面 | 塗料の比較、比較リストからの削除 |
| SC005 | 運用管理 | 運用に関する設定や管理を行う画面 | バッジ状態の確認、設定変更 |
